## 使用

首先你需要一个Python3.6的虚拟环境

1. 下载项目

```shell script
git clone https://github.com/janlle/12306.git 12306
cd 12306
```

2. 安装依赖

```shell script

python install -r requirements.txt

```

3. 启动你的应用

```shell script
python start.py
```
